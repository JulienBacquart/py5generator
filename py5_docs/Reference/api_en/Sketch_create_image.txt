@@ meta
name = create_image()
category = Image
subcategory = None
type = None
processing_name = createImage

@@ description
Creates a new PImage (the datatype for storing images). This provides a fresh buffer of pixels to play with. Set the size of the buffer with the ``width`` and ``height`` parameters. The ``format`` parameter defines how the pixels are stored. See the PImage reference for more information.
 
Be sure to include all three parameters, specifying only the width and height (but no format) will produce a strange error.
 
Advanced users please note that ``create_image()`` should be used instead of the syntax ``new p_image()``.

@@ example
image = createImage_0.png

Py5Image img = create_image(66, 66, RGB)
img.load_pixels()
for (int i = 0 i < img.pixels.length i++) {
  img.pixels[i] = color(0, 90, 102) 
}
img.update_pixels()
image(img, 17, 17)

@@ example
image = createImage_1.png

Py5Image img = create_image(66, 66, ARGB)
img.load_pixels()
for (int i = 0 i < img.pixels.length i++) {
  img.pixels[i] = color(0, 90, 102, i % img.width * 2) 
}
img.update_pixels()
image(img, 17, 17)
image(img, 34, 34)
