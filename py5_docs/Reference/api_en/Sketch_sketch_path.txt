where: str - subdirectories relative to the sketch path

@@ description
The Sketch's current path. Result is returned as a Python ``Path`` object.

This will be relative to ``os.getcwd()`` unless if you specifically set it to something else when calling :doc:`run_sketch` with a ``--sketch-path`` argument in the ``py5_options`` parameters.
