@@ meta
name = month()
category = Input
subcategory = Time & Date
type = None
processing_name = month

@@ description
Processing communicates with the clock on your computer. The ``month()`` function returns the current month as a value from 1 - 12.

@@ example
d = day()    # values from 1 - 31
m = month()  # values from 1 - 12
y = year()   # 2003, 2004, 2005, etc.

s = string.value_of(d)
text(s, 10, 28)
s = string.value_of(m)
text(s, 10, 56)
s = string.value_of(y)
text(s, 10, 84)
