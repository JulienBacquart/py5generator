## meta
name = width
category = Environment
subcategory = None
type = ['variable', 'variable']

## description
System variable that stores the width of the display window. This value is set by the first parameter of the <b>size()</b> function. For example, the function call <b>size(320, 240)</b> sets the <b>width</b> variable to the value 320. The value of <b>width</b> defaults to 100 if <b>size()</b> is not used in a program.

## example
image = width.png

noStroke();
background(0);
rect(0, 40, width, 20);
rect(0, 60, width/2, 20);
