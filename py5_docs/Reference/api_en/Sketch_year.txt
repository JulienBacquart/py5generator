@@ meta
name = year()
category = Input
subcategory = Time & Date
type = None
processing_name = year

@@ description
Processing communicates with the clock on your computer. The ``year()`` function returns the current year as an integer (2003, 2004, 2005, etc).

@@ example
int d = day()    int m = month()  int y = year()   
string s = string.value_of(d)
text(s, 10, 28)
s = string.value_of(m)
text(s, 10, 56) 
s = string.value_of(y)
text(s, 10, 84)
