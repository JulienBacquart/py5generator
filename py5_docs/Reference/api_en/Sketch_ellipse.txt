## meta
name = ellipse()
category = Shape
subcategory = 2D Primitives
type = None

## description
Draws an ellipse (oval) to the screen. An ellipse with equal width and height is a circle. By default, the first two parameters set the location, and the third and fourth parameters set the shape's width and height. The origin may be changed with the <b>ellipseMode()</b> function.

## example
image = ellipse_.png

ellipse(56, 46, 55, 55);
