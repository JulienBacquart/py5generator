@@ meta
name = end_camera()
category = Lights, Camera
subcategory = Camera
type = None
processing_name = endCamera

@@ description
The ``begin_camera()`` and ``end_camera()`` functions enable advanced customization of the camera space. Please see the reference for ``begin_camera()`` for a description of how the functions are used.

@@ example
image = Sketch_end_camera_0.png

size(100, 100, P3D)
no_fill()

begin_camera()
camera()
rotate_x(-PI/6)
end_camera()

translate(50, 50, 0)
rotate_y(PI/3)
box(45)
