@@ meta
name = text_size()
category = typography
subcategory = attributes
type = method
pclass = PApplet
processing_name = textSize

@@ signatures
text_size(size: float, /) -> None

@@ variables
size: float - the size of the letters in units of pixels

@@ description
Sets the current font size. This size will be used in all subsequent calls to the ``text()`` function. Font size is measured in units of pixels.

@@ example
image = Sketch_text_size_0.png

background(0)
fill(255)
text_size(26)
text("WORD", 10, 50)
text_size(14)
text("WORD", 10, 70)
