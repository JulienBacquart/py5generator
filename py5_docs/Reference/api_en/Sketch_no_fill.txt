## meta
name = noFill()
category = Color
subcategory = Setting
type = None

## description
Disables filling geometry. If both <b>noStroke()</b> and <b>noFill()</b> are called, nothing will be drawn to the screen.

## example
image = noFill_.png

rect(15, 10, 55, 55);
noFill();
rect(30, 20, 55, 55);
