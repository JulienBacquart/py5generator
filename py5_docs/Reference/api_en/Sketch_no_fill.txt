@@ meta
name = no_fill()
category = Color
subcategory = Setting
type = method
processing_name = noFill

@@ description
Disables filling geometry. If both ``no_stroke()`` and ``no_fill()`` are called, nothing will be drawn to the screen.

@@ example
image = Sketch_no_fill_0.png

rect(15, 10, 55, 55)
no_fill()
rect(30, 20, 55, 55)
