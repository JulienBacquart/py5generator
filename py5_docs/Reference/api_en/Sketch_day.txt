@@ meta
name = day()
category = Input
subcategory = Time & Date
type = None
processing_name = day

@@ description
Processing communicates with the clock on your computer. The ``day()`` function returns the current day as a value from 1 - 31.

@@ example
int d = day()    int m = month()  int y = year()   
string s = string.value_of(d)
text(s, 10, 28)
s = string.value_of(m)
text(s, 10, 56) 
s = string.value_of(y)
text(s, 10, 84)
